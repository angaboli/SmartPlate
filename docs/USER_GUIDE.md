# SmartPlate AI - Guide Utilisateur des Nouvelles Fonctionnalités

## 🌐 Changer la langue de l'interface

### Comment faire ?
1. Cliquez sur l'**icône globe** (🌐) en haut à droite du header
2. Sélectionnez votre langue préférée :
   - 🇬🇧 **English**
   - 🇫🇷 **Français**

### Qu'est-ce qui change ?
- Tous les menus, boutons et textes de l'interface sont traduits
- Le changement est instantané (pas de rechargement de page)
- Votre choix est mémorisé (localStorage) et retrouvé à votre prochaine visite, même après avoir fermé le navigateur

## 📥 Importer une recette depuis les réseaux sociaux

### Où trouver cette fonctionnalité ?
Vous pouvez importer une recette depuis **2 endroits** :

1. **Page d'accueil (Home)**
   - Bouton "Import Recipe" / "Importer une recette" dans la section principale
   
2. **Page Recettes (Recipes)**
   - Bouton "Import Recipe" en haut à droite, à côté du badge SafariTaste

### Comment importer une recette ?

#### Étape 1 : Copier le lien
- Allez sur Instagram, TikTok, YouTube ou n'importe quel site de recettes
- Copiez le lien de la recette que vous souhaitez sauvegarder

#### Étape 2 : Coller et analyser
1. Cliquez sur le bouton "Import Recipe"
2. Collez le lien dans le champ prévu
3. Cliquez sur "Fetch Recipe" / "Récupérer la recette"
4. Attendez quelques secondes pendant que l'IA analyse le lien

#### Étape 3 : Vérifier et éditer
L'application va automatiquement détecter :
- 📱 **Source** : Instagram, TikTok, YouTube ou Autre (avec icône)
- 🖼️ **Image** : Miniature de la recette
- 📝 **Titre** : Nom de la recette
- 👤 **Auteur** : Nom du créateur (ex: @foodlover_chef)
- ⏱️ **Temps de préparation** : Durée estimée
- 🥕 **Ingrédients** : Liste complète
- 📋 **Étapes** : Instructions de cuisson

**Si certaines informations manquent** :
- Un message vous informera que certains détails n'ont pas pu être détectés
- Vous pouvez les ajouter manuellement dans les champs d'édition

#### Étape 4 : Taguer et sauvegarder
1. (Optionnel) Sélectionnez un tag de repas :
   - 🌅 Breakfast / Petit-déjeuner
   - 🌞 Lunch / Déjeuner
   - 🌙 Dinner / Dîner
   - 🍎 Snack / Collation

2. Cliquez sur "Add to Cook Later" / "À cuisiner plus tard"
3. Une notification confirmera que votre recette est sauvegardée ✅

## 🍳 Gérer vos recettes "Cook Later" (À cuisiner plus tard)

### Où trouver mes recettes sauvegardées ?
1. Allez dans **Profile** (Profil)
2. Cliquez sur l'onglet **"Cook Later"** / **"À cuisiner"**

### Que puis-je faire avec mes recettes sauvegardées ?

Chaque recette affiche :
- ✅ Miniature avec effet de survol
- 📝 Titre et source (avec badge coloré)
- 👤 Auteur (si disponible)
- ⏱️ Temps de préparation
- 📅 Date d'ajout
- 🏷️ Tag de repas (si défini)
- 🥕 Aperçu des ingrédients

#### Actions disponibles :

**1. Open Recipe / Ouvrir la recette** 🔗
- Ouvre le lien original dans un nouvel onglet
- Pratique pour voir la recette complète sur la source

**2. Add to Planner / Ajouter au planning** 📅
- (Fonctionnalité future) Ajoutera la recette à votre planificateur hebdomadaire

**3. Mark as Cooked / Marquer comme cuisinée** ✅
- Marque la recette comme préparée
- La carte devient semi-transparente avec une icône de validation
- Permet de suivre ce que vous avez déjà cuisiné

**4. Remove / Retirer** 🗑️
- Supprime définitivement la recette de votre liste
- Attention : cette action ne peut pas être annulée

### État vide
Si vous n'avez pas encore de recettes sauvegardées, vous verrez :
- 👨‍🍳 Une icône de chef
- Un message vous invitant à importer votre première recette
- Un lien rapide vers la fonctionnalité d'import

## 📌 Navigation Sticky (Menu fixe)

### C'est quoi ?
Le sous-menu principal **Home | AI Coach | Recipes | Profile** reste maintenant **toujours visible** en haut de l'écran, même quand vous scrollez vers le bas.

### Avantages :
- ✅ Navigation rapide entre les pages sans remonter en haut
- ✅ Vous savez toujours où vous êtes (page active surlignée)
- ✅ Gain de temps et meilleure expérience utilisateur

### Sur mobile :
- Le menu devient scrollable horizontalement si nécessaire
- Pas de scrollbar visible (design épuré)
- Accès au menu burger (☰) à gauche pour navigation complète

## 💡 Conseils et Astuces

### Pour une meilleure expérience :

1. **Importez des recettes variées**
   - Utilisez les tags (Breakfast, Lunch, Dinner, Snack) pour organiser
   - Importez depuis différentes sources pour diversifier

2. **Marquez les recettes cuisinées**
   - Gardez un historique de ce que vous avez préparé
   - Identifiez rapidement les recettes non testées

3. **Utilisez le mode sombre**
   - Icône 🌙/☀️ en haut à droite pour changer
   - Idéal pour cuisiner le soir sans éblouissement

4. **Combinez avec les recettes SafariTaste**
   - Vos recettes importées complètent le catalogue existant
   - Créez votre propre bibliothèque culinaire personnalisée

## ❓ FAQ

### Q : Quelles plateformes sont supportées pour l'import ?
**R :** Instagram, TikTok, YouTube et tout autre lien public de recette. L'application détecte automatiquement la source.

### Q : Combien de recettes puis-je sauvegarder ?
**R :** Il n'y a pas de limite ! Sauvegardez autant de recettes que vous le souhaitez.

### Q : Les recettes sont-elles synchronisées entre appareils ?
**R :** Oui. Vos recettes sauvegardées sont liées à votre compte et stockées en base de données — connectez-vous sur n'importe quel appareil pour les retrouver.

### Q : Puis-je modifier une recette après l'avoir importée ?
**R :** Oui ! Lors de l'import, vous pouvez éditer tous les champs avant de sauvegarder.

### Q : Que se passe-t-il si un lien devient invalide ?
**R :** Le bouton "Open Recipe" vous informera si le lien n'est plus accessible, mais votre recette sauvegardée restera visible avec toutes les infos extraites.

### Q : Puis-je importer des recettes en plusieurs langues ?
**R :** Oui ! Le contenu de la recette (titre, ingrédients, étapes) est conservé dans sa langue d'origine.

## 🆘 Besoin d'aide ?

Si vous rencontrez un problème :
1. Vérifiez que le lien est bien public (pas de contenu privé)
2. Essayez de recharger la page
3. Changez de langue et revenez à votre langue préférée
4. Contactez le support SmartPlate AI
